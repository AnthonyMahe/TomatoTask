## üöÄ Usage

### Getting Started

1. **Start the Timer** - Click the timer button or press `Ctrl+S`
2. **Create Tasks** - Click "New Task" or press `Ctrl+N`
3. **Organize with Projects** - Click "+" in the projects panel to create a project
4. **Track Progress** - View your daily and weekly statistics

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Start/Stop timer |
| `Ctrl+N` | New task |
| `Ctrl+L` | Change language |
| `Ctrl+/` | Show all shortcuts |
